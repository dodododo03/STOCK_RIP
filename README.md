STOCK_RIP 프로젝트 설명
1. 프로젝트 컨셉: "내 계좌, 잠들다 (RIP)"

화려한 주식 분석이 아니라, "나만 망한 거 아니지?"를 확인하며 위안을 얻는 주식 낙폭/손실 공유 커뮤니티.

나스닥 고점에 물린 분들이 모여서 눈물 흘리는 공간. (본인 얘기는 아님..)

2. 구현된 코드의 실체 (현재 단계)

데이터 수집: yfinance로 주가를 가져오긴 하지만, 이건 수익률 계산용이 아니라 "얼마나 처참하게 박살 났나"를 실시간으로 확인하기 위한 용도.

백엔드 구조: FastAPI를 사용해 "나 오늘 -30% 찍었다" 같은 글을 올릴 수 있는 게시판의 기초 뼈대(CRUD).

데이터베이스: 주주들의 한 서린 게시글과 댓글을 저장하기 위한 SQLAlchemy 기반의 테이블 설계.

3. 핵심 기능 (작업 중)

방명록(댓글) 기능: "형님 기운 내세요", "여기 동지 한 명 추가요" 같은 댓글을 달 수 있는 소통 창구.
